Introdução a MVC

O padrão MVC (Model-View-Controller) é uma arquitetura de software utilizada no desenvolvimento de aplicações para separar a lógica de negócios, a interface do usuario e o controle dos dados.

Componentes MVC

  • Model (Modelo) - Representa os dados e a lógica de negócio da aplicação.
  • View (Visão) - Responsavel pela interface com o usuario e a exibição dos dados.
  • Controller (Controlador) - Intermedia a comunicação entre Model e View.
